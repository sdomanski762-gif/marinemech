# MarineMech App

MarineMech is an offline-first web application designed for marine diagnostics. It helps users troubleshoot issues with diesel engines, gasoline outboards, and marine electrical systems through interactive diagnostic trees and expert mode analytics.

## Features
- **Interactive Diagnostic Trees**: Step-by-step troubleshooting for common marine failures.
- **Offline Capable**: Works without an internet connection once loaded.
- **Marine Specifics**: Custom logic for Yanmar, Volvo Penta, Yamaha, Mercury, and Honda engines.
- **Electrical Safety**: Built-in protocols for handling smoke, sparks, and fire hazards.
- **Amateur & Expert Modes**: Tailored advice based on the user's technical level.

## Project Structure
- `FOUNDATIONV1.html`: The main single-page application.
- `marine_specifics.js`: The core logic and knowledge base.
- `diagnostic_paths/`: Modular diagnostic logic files.
- `tests/`: Validation scripts for the diagnostic logic.
- `SCHEMA.md`: Documentation for the diagnostic tree data contract.

## Usage
Simply open `FOUNDATIONV1.html` in any modern web browser.

## Mobile Access
This repository allows you to access and work on the diagnostic logic from your phone via GitHub.
